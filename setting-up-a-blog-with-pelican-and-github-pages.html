Setting up a blog with Pelican and Github Pages

Sat 10 April 2021 in dev
#pelican #blog

This is about how this blog was setup using Pelican and Github Pages. I anticipate refining this process so will keep this page updated with an improvements made along the way.

Why Github Pages

Basically: - it's free - I already use Github for most of my projects - Endless customisation options - it's a static site, HTML, CSS and JavaScript FTW

Why Pelican

So if you follow the tutorial on setting up Github pages you'll notice that the preferred site generation uses Jekyll. Jekyll is a nice option, particularly in terms of the themes available.
