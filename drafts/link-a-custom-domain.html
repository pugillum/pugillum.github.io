Link a custom domain

Sun 25 April 2021 in dev
#github pages #pelican #blog

When you first setup a site using Github pages your site URL will be something like [bla bla].github.io. You may want to purchase and link your URL. This is about the process of doing that, though there is also documentation available from Github here

Provisional structure: - requesting a URL from Porkbun - reasonable price - renewal

  • configuring URL mapping in Porkbun [https://kb.porkbun.com/article/85-how-to-connect-your-root-domain-when-your-web-host-wont-provide-an-ip-address]

  • Github settings

    • waiting before turning on DNS

  • Adjustment to publish to change URL in publishconf.py

  • sort out the git thing

  • Including CNAME STATIC_PATHS = ['images', 'extra/CNAME']

Important

If your links in your publish are pugillum.github.io you'll need to change them You can't enable HTTPS straight away
